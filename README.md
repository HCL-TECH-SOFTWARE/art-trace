**art-trace**

Utilities for parsing, analyzing and translating .art-trace files. Such files can be generated by realtime applications developed with [Code RealTime](https://secure-dev-ops.github.io/code-realtime/) or [Model RealTime](https://model-realtime.hcldoc.com/help/topic/com.ibm.xtools.rsarte.webdoc/users-guide/overview.html?cp=23_0).
